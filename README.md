# EDUPI4.0 - Low-Cost Educational PLC for Industry 4.0

EDUPI4.0 is a modular, low-cost Programmable Logic Controller (PLC) designed for technical education in industrial automation and Industry 4.0 environments.

## Key Features

- Based on Raspberry Pi Zero 2W
- Compatible with IEC 61131-3 (CODESYS)
- Digital inputs (24V opto-isolated)
- Relay outputs (up to 230VAC)
- Modular architecture
- Integration with IoT technologies (MQTT, Node-RED)

## Cost

The total electronic cost per unit is below **30€** for a batch of 10 units, including:
- Raspberry Pi Zero 2W
- Custom PCB
- Electronic components

See `/hardware/BOM/` for detailed cost breakdown.
